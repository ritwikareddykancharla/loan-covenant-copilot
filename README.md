## ğŸ§  What is Covenant Monitoring? (Context for Judges)

A *covenant* is a promise made by a borrower to a lender in a loan agreement.
Covenant monitoring ensures these promises are continuously met throughout the life of the loan.

The **Loan Market Association (LMA)** provides standardized loan documentation and best-practice covenant language, but **monitoring remains largely manual** across the industry.
